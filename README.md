<h1>My config neovim file</h1>

<h2>Install:</h2>

```
mkdir -p ~/.config ~/.vim && \
mkdir -p ~/.vim/bundle ~/.vim/autoload && \
ln -s ~/.vim ~/.config/nvim && \
cd ~ && \
git clone https://github.com/minhbk/Neovim-configuration.git && \
ln -s  ~/Neovim-configuration/vimrc ~/.config/nvim/init.vim
```
